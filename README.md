## ✨ Features

| Feature | Description |
|---------|-------------|
| 📊 **Admin Dashboard** | Daily summary of milk collected, sold, payments received and expenses with trend indicators |
| 👥 **Customer Management** | Add and manage customers with daily milk quantity, contact info, and real-time balance tracking |
| 🥛 **Milk Production Tracking** | Log primary milk production and milk sales with quick-action shortcuts |
| 💰 **Payment & Ledger System** | Track payments received, expenses, and pending approvals with full transaction history |
| 📈 **Reports & Analytics** | 7-day financial activity chart showing milk production and revenue trends over time |
| 👤 **Role-Based Access** | Separate staff and admin views with personalized dashboards and access permissions |

## 🚀 Getting Started

### Prerequisites
- Flutter SDK (3.0+)
- Dart SDK
- A Supabase account and project

### Installation

```bash
# Clone the repository
git clone https://github.com/malibharion/MilkMate.git
cd MilkMate

# Install dependencies
flutter pub get

# Run the app
flutter run
```

> ⚠️ **Note:** Add your own `supabase_url` and `supabase_anon_key` to the Supabase config file before running.
